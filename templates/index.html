Welcome to the Student Exam Performance Predictor

This project uses a dataset containing information about students’ backgrounds and test scores. It includes features such as gender, ethnicity, parental education level, lunch type, test preparation course, and performance in reading and writing. The goal is to predict a student’s mathematics score based on these inputs.

By exploring this dataset, we can better understand how different factors influence student achievement and build models that estimate exam performance with higher accuracy.
